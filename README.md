# title:  
connectinn  
# description:  
旅行者と民泊提供者をマッチングするためのサービスです。  
サービスの魅力はマッチングのプラットフォーム提供、マッチング後のチャット機能、住所情報の加工機能（県、市などのみ表示）など互いのプライベートな情報を守りつつ、信頼関係の構築を図ります。
